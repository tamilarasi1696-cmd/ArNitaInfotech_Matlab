# MATLAB/Simulink Internship Projects – Anita InfoTech

## Overview

This repository contains the tasks completed as part of my MATLAB/Simulink Internship at Anita InfoTech. The internship focused on modeling, simulation, analysis, and control of a DC Motor using MATLAB and Simulink.

## Internship Domain

* MATLAB
* Simulink
* Control Systems
* DC Motor Modeling
* PID Controller Design

## Tasks Completed

### Task 1: Mathematical Modeling of DC Motor

* Developed the mathematical model of a DC motor.
* Derived the transfer function based on motor parameters.
* Analyzed the motor dynamics.

### Task 2: DC Motor Simulation in Simulink

* Created a Simulink model of the DC motor.
* Simulated motor behavior under different operating conditions.
* Verified the mathematical model through simulation.

### Task 3: Speed Control of DC Motor Using PID Controller

* Designed a PID controller for speed regulation.
* Tuned PID parameters using PID Tuner.
* Analyzed system performance such as rise time, settling time, and overshoot.

### Task 4: Overall Simulation and Demonstration

* Integrated the complete DC motor control system.
* Demonstrated system performance through simulation videos.
* Evaluated controller effectiveness and motor response.
